# Algo Harvest 🌱

Algo Harvest is a spaced repetition system designed for long-term algorithm retention.

Instead of endlessly re-solving coding problems, Algo Harvest helps developers reinforce algorithmic patterns over time through scheduled reviews, memory-strength tracking, and progression-based learning.

Built with Angular, TypeScript, and Tailwind CSS, the project combines interview preparation with a cozy growth-oriented experience inspired by farming and retro progression systems.

## Features

* Track solved algorithm problems
* Spaced repetition review scheduling
* Harvest/review workflow
* Memory-strength based progression
* Local persistence with localStorage
* Responsive modern UI
* Deployed omean the tiny den Vercel

## Vision

Solve problems once.
Remember them long term.
Grow your algorithmic memory like a garden.
